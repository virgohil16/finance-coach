# AI-Powered Personal Finance Coach

An intelligent personal finance application that analyses bank transaction data to deliver personalised spending insights, automated categorisation, and AI-generated savings recommendations.

## Overview

This project combines transaction analytics with large language models to help users understand their spending habits and make smarter financial decisions. Users upload their bank statements, and the system automatically categorises transactions, identifies wasteful patterns, and provides actionable advice through a conversational interface.

## Key Features

- **Automated transaction categorisation** using Llama 3 via the Groq API
- **Spending pattern analysis** with category-level breakdowns and trend detection
- **AI-generated coaching insights** providing personalised savings recommendations
- **Interactive Streamlit dashboard** featuring spending visualisations and filters
- **SQL-backed transaction history** for long-term tracking and analysis

## Tech Stack

- **Language:** Python 3.11
- **AI/ML:** Llama 3 (Groq API), prompt engineering
- **Data:** pandas, NumPy
- **Database:** PostgreSQL / SQLite
- **Frontend:** Streamlit, Plotly
- **Version Control:** Git, GitHub
